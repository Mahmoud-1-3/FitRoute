5.4 Bug Report

Bug ID Module Issue Description Root Cause Resolution Status
BUG-001 Routing App gets stuck in an infinite redirect loop on startup. GoRouter was trying to check the user's role before the Hive local database had fully initialized. Modified main.dart to await Hive.initFlutter() before attaching the router to the UI. Resolved
BUG-002 AI Scanner "Rate My Plate" randomly crashes and shows a red error screen. Gemini API occasionally returned the JSON wrapped in Markdown formatting, crashing the Dart JSON parser. Updated the GenerationConfig to explicitly enforce responseMimeType: 'application/json'. Resolved
BUG-003 Performance Capturing a photo for the AI scanner takes 10+ seconds to process. High-resolution modern smartphone cameras were creating 10MB+ files, causing slow API uploads. Implemented imageQuality: 50 and maxWidth: 800 in the image_picker to compress files under 1MB. Resolved
